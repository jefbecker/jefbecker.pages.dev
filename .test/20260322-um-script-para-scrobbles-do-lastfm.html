Um Script Para Scrobbles do Last.FM

22 Mar 20236 | Domingo | 17:00

Em algum momento de 2024, eu procurei por um script que exibisse as músicas do Last.FM no meu site com mais informações que o código da @biancarosa. O script dela é ótimo, só que ele mostra apenas o nome da música e eu queria exibir a capa do álbum. Por acaso, encontrei no fórum do last.fm uma pessoa pedindo ajuda com um plugin do wordpress e alguém postou um script completo que funciona em qualquer site.

Começando

Em primeiro lugar, o script está disponível aqui, se quiser fazer tudo por conta própria. Mas se quiser seguir o passo a passo com a versão com as minhas alterações, veja a seguir.

Você vai precisar de três coisas:

  • Que o seu site permita conexão remota;
  • Uma conta com acesso à API do Last.FM;
  • Do script, obviamente.

Conexão Remota

Esse á a parte mais importante, sem ela seu script pode não funcionar. Contas gratuitas e/ou recentemente criadas no Neocities não podem mais carregar conteúdo remoto de outros sites. Segundo Kyle, essa função foi limitada para impedir abuso. Sites criados até o final de 2023 podem ainda ter essa opção. Sites com plano pago tem essa função liberada desde sempre.

Só que fora do Neocities, as principais hospedagens de sites estáticos oferecem essa função mesmo nos planos gratuitos. É o caso do Github Pages, Cloudflare Pages e Nekoweb. Com certeza existem outros, mas eu desconheço.

API do Last.FM

Para que o script funcione, também será necessário criar uma chave API no Last.FM. Nessa página você deverá preencher um email de contato, o nome, uma descrição e a página da aplicação. Os dados desse formulário não importam muito, embora eu recomende que use o mesmo email da sua conta no Last.FM, só pra garantir.

Ao concluir, você terá a sua chave criada na página de contas de API, onde é possível consultar e revogar todas a chaves criadas.

Script

Você pode copiar o script abaixo com as minhas alterações. São elas: traduções de termos para o português, exibir a capa do album maior, localização de datas e do horário para a região do Brasil e ajustes de layout para o meu site.

A última linha do script é onde irão as alterações de acordo com os seus dados. Você vai colar a sua chave API no lugar de "SUA CHAVE API", seu nome de usuário do Last.FM no lugar de "USUARIO" e o id que você preferir usar no seu HTML no lugar de "ID HTML"

Abaixo está detalhada a parte do script que contém elementos que podem ser customizados para atender ao estilo do seu site. Só tome cuidado porque alguns elementos fazem parte do código.

Por fim, basta colocar o "id" que você preferir no seu arquivo html e o link do script. No meu caso, uso o id "lastfm_en": <div id="lastfm_en" style="min-height:6.25em; margin-bottom: -0.7em;"></div>. Você pode usar só o "id", mas é interessante usar também um tamanho mínimo pro elemento para evitar mudanças de layout ao carregar a página style="min-height:6.25em.
